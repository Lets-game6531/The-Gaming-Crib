Java vs Bedrock Edition: What’s Different on Our Server?

Our Minecraft community welcomes players from both the Java and Bedrock editions, and we’ve worked hard to support smooth crossplay. Here’s what you need to know:

Gameplay Differences

  • Java Edition: Offers extensive modding support, classic redstone mechanics, and is typically played on PC or Mac.
  • Bedrock Edition: Available on consoles, mobile devices, and Windows 10/11, with optimized performance and different UI layout.

Crossplay on Our Server

Using Floodgate and Geyser plugins, we allow Bedrock players to join our Java-based server seamlessly. However, some features may vary slightly:

  • Skins and capes might look different between editions.
  • Bedrock players have some interface limitations.
  • Redstone and command blocks may behave differently.
